# Python 서비스 추가 - 빠른 참조

새로운 Python 서비스를 `services/python`에 추가하는 빠른 가이드입니다.

---

## 빠른 시작 (5분)

서비스 이름: `notification` (예시)

```bash
cd /workspace/services/python

# 1. 디렉토리 생성
mkdir -p notification/src notification/tests
touch notification/src/__init__.py
touch notification/tests/__init__.py

# 2. pyproject.toml 생성
cat > notification/pyproject.toml <<'EOF'
[project]
name = "notification"
version = "0.1.0"
description = "Notification service"
requires-python = ">=3.10"

dependencies = [
    "fastapi>=0.109.0",
    "uvicorn>=0.27.0",
    "pydantic>=2.5.0",
]

[project.optional-dependencies]
dev = [
    "pytest>=7.4.0",
    "pytest-asyncio>=0.23.0",
]

[build-system]
requires = ["setuptools>=68.0"]
build-backend = "setuptools.build_meta"

[tool.setuptools.packages.find]
where = ["."]
include = ["src*"]
EOF

# 3. BUILD 파일 생성
cat > notification/BUILD <<'EOF'
python_sources(
    name="src",
    sources=["src/**/*.py"],
)

python_tests(
    name="tests",
    sources=["tests/**/*.py"],
    dependencies=[":src"],
)

pex_binary(
    name="notification",
    entry_point="src.main:main",
    dependencies=[":src"],
)
EOF

# 4. main.py 작성
cat > notification/src/main.py <<'EOF'
"""Notification service."""

from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI(title="Notification Service")

class HealthResponse(BaseModel):
    status: str
    service: str

@app.get("/health")
async def health() -> HealthResponse:
    return HealthResponse(status="healthy", service="notification")

def main() -> None:
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8085)

if __name__ == "__main__":
    main()
EOF

# 5. 테스트 작성
cat > notification/tests/test_main.py <<'EOF'
"""Tests for notification service."""

from fastapi.testclient import TestClient
from src.main import app

client = TestClient(app)

def test_health():
    response = client.get("/health")
    assert response.status_code == 200
    data = response.json()
    assert data["status"] == "healthy"
EOF

# 6. Pants 의존성 재생성
cd /workspace
./pants generate-lockfiles

# 7. 빌드 및 테스트
./pants package services/python/notification:notification
./pants test services/python/notification::

# 8. 실행 테스트
./dist/notification.pex &
curl http://localhost:8085/health
# 출력: {"status":"healthy","service":"notification"}

# 프로세스 종료
pkill -f notification.pex
```

## 빠른 테스트

### 빌드

```bash
cd /workspace

# 특정 서비스 빌드
./pants package services/python/notification:notification

# 결과 확인
ls -lh dist/notification.pex

# 모든 Python 서비스 빌드
./pants package services/python::
```

### 테스트

```bash
# 전체 테스트
./pants test services/python/notification::

# 커버리지 포함
./pants test --test-use-coverage services/python/notification::

# 특정 테스트 함수
./pants test services/python/notification/tests/test_main.py::test_health
```

### 린트 & 포맷

```bash
# Ruff 린트 (체크만)
./pants lint services/python/notification::

# MyPy 타입 체크
./pants check services/python/notification::

# 자동 포맷 (문법적 수정)
./pants fmt services/python/notification::

# 자동 수정 (의미적 수정 - Pants 2.30.1+)
./pants fix services/python/notification::
```

### 실행

```bash
# PEX 실행
./dist/notification.pex

# 백그라운드 실행
./dist/notification.pex &

# Makefile로 실행 (추가 시)
make python-dev-notification

# 헬스 체크
curl http://localhost:8085/health
```

./pants -ldebug package services/python/notification:notification

## 공유 패키지 사용

### 1. 공유 패키지 생성

```bash
mkdir -p shared/python/common
touch shared/python/common/__init__.py

cat > shared/python/common/BUILD <<'EOF'
python_sources(
    name="common",
    sources=["**/*.py"],
)
EOF

cat > shared/python/common/types.py <<'EOF'
from pydantic import BaseModel

class ServiceResponse(BaseModel):
    success: bool
    message: str
    data: dict | None = None
EOF
```

### 2. 서비스에서 사용

**BUILD 파일:**

```python
python_sources(
    name="src",
    sources=["src/**/*.py"],
    dependencies=[
        "shared/python/common:common",  # 추가
    ],
)
```

**Python 코드:**

```python
from shared.python.common.types import ServiceResponse

@app.get("/status")
async def status() -> ServiceResponse:
    return ServiceResponse(
        success=True,
        message="Service running",
    )
```

## FastAPI 엔드포인트 템플릿

### Health Check

```python
from pydantic import BaseModel

class HealthResponse(BaseModel):
    status: str
    service: str
    version: str

@app.get("/health")
async def health() -> HealthResponse:
    return HealthResponse(
        status="healthy",
        service="notification",
        version="0.1.0",
    )
```

### POST 엔드포인트

```python
class CreateRequest(BaseModel):
    name: str
    description: str | None = None

class CreateResponse(BaseModel):
    id: str
    name: str
    created_at: str

@app.post("/create")
async def create(request: CreateRequest) -> CreateResponse:
    # 비즈니스 로직
    return CreateResponse(
        id="123",
        name=request.name,
        created_at="2026-03-09T00:00:00Z",
    )
```

### 에러 핸들링

```python
from fastapi import HTTPException

@app.post("/process")
async def process(data: dict):
    if not data:
        raise HTTPException(
            status_code=400,
            detail="Data is required",
        )
    # 처리 로직
```

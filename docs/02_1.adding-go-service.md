## 디렉토리 구조 이해

### 현재 구조

```
services/go/
├── go.work                  # Go workspace 설정 (모든 모듈 관리)
├── pkg/                     # 공유 라이브러리
│   ├── go.mod
│   ├── common/
│   └── utils/
├── gateway/                 # Gateway 서비스
│   ├── BUILD               # Pants 빌드 파일 (사용 안 함)
│   ├── go.mod              # 모듈 정의
│   ├── cmd/
│   │   └── gateway/
│   │       └── main.go     # 엔트리포인트
│   └── internal/
│       └── handler/
└── detection/               # Detection 서비스
    ├── BUILD
    ├── go.mod
    ├── cmd/
    │   └── detection/
    │       └── main.go
    └── internal/
        └── detector/
```

### Go Workspace 구조

이 모노레포는 **Go Workspace**를 사용합니다:

- `services/go/go.work`: 모든 Go 모듈을 관리하는 workspace 파일
- 각 서비스는 독립적인 `go.mod`를 가짐
- `pkg`는 공유 라이브러리로 모든 서비스에서 사용 가능

## 새 서비스 추가 단계

### 1단계: 디렉토리 생성

새 서비스를 위한 디렉토리 구조를 생성합니다.

**예: `analytics` 서비스 추가**

```bash
cd /workspace/services/go

# 서비스 디렉토리 생성
mkdir -p analytics/cmd/analytics
mkdir -p analytics/internal/processor

# 기본 파일 생성
touch analytics/cmd/analytics/main.go
touch analytics/internal/processor/processor.go
```

### 2단계: go.mod 파일 생성

각 서비스는 독립적인 Go 모듈입니다.

```bash
cd analytics

# go.mod 초기화
go mod init github.com/example/monorepo/services/go/analytics
```

**`analytics/go.mod` 내용:**

```go
module github.com/example/monorepo/services/go/analytics

go 1.23

// 공유 패키지를 사용할 경우
require github.com/example/monorepo/services/go/pkg v0.0.0
```

> **참고**: `go.work`를 사용하므로 `replace` 지시문은 필요 없습니다.

### 3단계: go.work에 추가

workspace에 새 모듈을 등록합니다.

```bash
cd /workspace/services/go

# go.work 파일 수정
```

**`services/go/go.work` 수정:**

```go
go 1.22

use (
	./gateway

	./pkg

)
```

또는 자동으로 추가:

### 2. go.mod 생성

### 3. go.work 업데이트


### 4. 소스 코드 작성

**main.go:**

```bash
cat > analytics/cmd/analytics/main.go <<'EOF'
package main

import (
	"encoding/json"
	"fmt"
	"log"
	"net/http"
	"time"
)

type Metric struct {
	Name      string    `json:"name"`
	Value     float64   `json:"value"`
	Timestamp time.Time `json:"timestamp"`
}

type AnalyticsService struct {
	metrics []Metric
}

func NewAnalyticsService() *AnalyticsService {
	return &AnalyticsService{
		metrics: make([]Metric, 0),
	}
}

func (s *AnalyticsService) healthHandler(w http.ResponseWriter, r *http.Request) {
	w.WriteHeader(http.StatusOK)
	fmt.Fprintf(w, "Analytics service is healthy\n")
}

func (s *AnalyticsService) metricsHandler(w http.ResponseWriter, r *http.Request) {
	w.Header().Set("Content-Type", "application/json")
	json.NewEncoder(w).Encode(s.metrics)
}

func (s *AnalyticsService) recordHandler(w http.ResponseWriter, r *http.Request) {
	if r.Method != http.MethodPost {
		http.Error(w, "Method not allowed", http.StatusMethodNotAllowed)
		return
	}

	var metric Metric
	if err := json.NewDecoder(r.Body).Decode(&metric); err != nil {
		http.Error(w, err.Error(), http.StatusBadRequest)
		return
	}

	metric.Timestamp = time.Now()
	s.metrics = append(s.metrics, metric)

	w.WriteHeader(http.StatusCreated)
	json.NewEncoder(w).Encode(metric)
}

func main() {
	service := NewAnalyticsService()

	http.HandleFunc("/health", service.healthHandler)
	http.HandleFunc("/metrics", service.metricsHandler)
	http.HandleFunc("/record", service.recordHandler)

	port := ":8084"
	log.Printf("Analytics service starting on %s", port)

	if err := http.ListenAndServe(port, nil); err != nil {
		log.Fatal(err)
	}
}
EOF
```

**processor.go:**

```bash
cat > analytics/internal/processor/processor.go <<'EOF'
package processor

import (
	"log"
	"time"
)

type Event struct {
	Type      string
	Data      map[string]interface{}
	Timestamp time.Time
}

type Processor struct {
	events []Event
}

func New() *Processor {
	return &Processor{
		events: make([]Event, 0),
	}
}

func (p *Processor) ProcessEvent(eventType string, data map[string]interface{}) error {
	event := Event{
		Type:      eventType,
		Data:      data,
		Timestamp: time.Now(),
	}

	p.events = append(p.events, event)
	log.Printf("Processed event: %s at %s", eventType, event.Timestamp)

	return nil
}

func (p *Processor) GetEvents() []Event {
	return p.events
}
EOF
```

**processor_test.go:**

```bash
cat > analytics/internal/processor/processor_test.go <<'EOF'
package processor

import (
	"testing"
)

func TestProcessor_ProcessEvent(t *testing.T) {
	p := New()

	data := map[string]interface{}{
		"user_id": "123",
		"action":  "click",
	}

	err := p.ProcessEvent("user_action", data)
	if err != nil {
		t.Errorf("ProcessEvent() error = %v", err)
	}

	events := p.GetEvents()
	if len(events) != 1 {
		t.Errorf("Expected 1 event, got %d", len(events))
	}

	if events[0].Type != "user_action" {
		t.Errorf("Expected event type 'user_action', got '%s'", events[0].Type)
	}
}
EOF
```
